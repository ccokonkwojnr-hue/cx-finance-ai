## What is CX FINANCE AI?

CX FINANCE AI is a concept product and portfolio project that demonstrates how agentic AI workflows can be applied to finance education. It combines:

- **Autonomous research agents** that scan market data, news, and on-chain signals
- **Paper-trading simulations** — strategy testing with virtual money, no real execution
- **Signal education** — plain-English explanations of RSI, MACD, order blocks, and more
- **Risk literacy tools** — stress tests, portfolio scenario analysis, and scam-awareness education

> ⚠️ **Important:** CX FINANCE AI is not a regulated broker or investment adviser. It does not execute real trades or hold client funds. All outputs are for education and paper-trading only using virtual money.

## Design Decisions

### Why cream + teal + gold?
The dark/purple crypto exchange aesthetic was deliberately avoided. Finance education tools need to feel calm, trustworthy, and accessible — closer to Bloomberg or a premium wealth management dashboard than a DeFi platform.

### Why paper-trading only?
Regulatory clarity. Any tool that executes real trades, holds funds, or gives personalised investment advice needs FCA authorisation in the UK. By scoping to education and simulation only, CX FINANCE AI stays clearly outside regulated territory while still being genuinely useful.

### Why agentic workflows?
The three-agent architecture (SNAP / RESEARCH / RISK) demonstrates how multi-step AI reasoning can be structured and explained to non-technical users — a core skill for building AI products that real people can trust.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app: `npm run dev`
